# Sofimar SERV - Website

Site-ul oficial al Sofimar SERV - Servicii profesionale de Deratizare, Dezinsecție și Dezinfecție.

## Tehnologii

- HTML5
- CSS3 (cu efecte glassmorphism și 3D)
- JavaScript (vanilla)
- TikTok Embed API

## Structura Proiectului

```
site/
├── index.html          # Pagina principală
├── certificate.html    # Pagina cu certificate
├── styles.css          # Stiluri CSS
├── script.js           # JavaScript
├── images/             # Imagini și certificate
└── videos/             # Videoclipuri (nu sunt incluse în git)
```

## Caracteristici

- Design modern cu efecte glassmorphism
- Secțiune TikTok cu carousel automat
- Chatbot interactiv
- Secțiune certificate dedicată
- Design responsive
- Video cinematic ca fundal

## Instalare

1. Clonează repository-ul
2. Deschide `index.html` în browser

## Notă

Videoclipurile din folderul `videos/` nu sunt incluse în repository din cauza dimensiunii lor. Adaugă-le manual după clonare.
